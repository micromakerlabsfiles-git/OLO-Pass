Welcome to OLO Pass

A secure, air-gapped BLE password vault and keyboard emulator for your ESP32. Connect via USB Serial to manage passwords, configure settings, and control your device.

📖 Quick Start Guide

🤔 What is OLO Pass?

  • Password Vault — stores credentials securely on the device itself (not the cloud).
  • BLE Keyboard — wirelessly types credentials into any device.
  • OLED Face UI — animated eye display for feedback and navigation.
  • Web Controller — this page! Manage everything over USB.

⚡ System Requirements

  • Google Chrome or Microsoft Edge browser (Web Serial API).
  • OLO Bit ESP32-C3 board flashed with OLO Pass firmware.
  • USB-C cable for configuration; Bluetooth for auto-type.

🔌 Step 1 — Connect via USB

  1. Plug your OLO Pass device into your computer using a USB-C cable.
  2. Click ⚡ Connect via USB Serial above.
  3. A browser pop-up will show available serial ports — select your device (usually labelled USB Serial or CP210x).
  4. The device screen will show a 4-digit OTP. Enter it in the Web Controller to unlock the dashboard.
🔑 The OTP changes every session for security. It is shown only on the physical OLED screen.

🔑 Step 2 — Manage Your Vault

  1. Go to the Vault Manager tab after unlocking.
  2. Fill in the Title, Username/Email, Password, and optional URL in the form.
  3. Click Save to Device to store the record on the hardware.
  4. To edit or delete a record, click Edit next to any entry.

📡 Step 3 — Auto-Type with BLE

  1. On the OLO Pass device, pair it as a Bluetooth keyboard from your phone or computer.
  2. Navigate to a login page and focus the username field.
  3. On the OLO Pass device, navigate to Password Vault, select your account.
  4. Single-tap to type the Username, double-tap to type the Password.

Information & Guide
  • 🔒 Direct Keyboard Injection: Text pushed from here is typed directly onto your BLE connected device as if you were typing on a physical keyboard.
  • 📝 Multiline Support: Newlines are fully supported. Use this to push scripts, code blocks, or standard messages.
  • ⚡ OS Layout Compatibility: Ensure your OLO Pass OS settings match the target operating system for proper typing speed.
  • 🔋 Status Indicator: Check the device screen or the status bar above to confirm Bluetooth connection before pushing.
